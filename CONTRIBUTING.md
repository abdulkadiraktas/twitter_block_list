# Katki Rehberi

Bu depo sadece blocklist.json dosyasını barındırır. Katkılar basit, küçük ve denetlenebilir olmalı.

## Değişiklik Kuralları

- accounts listesi küçük harf olmalıdır
- @ işareti kullanılmaz
- Tekrarları eklemeyin
- Her değişiklikte version değerini 1 artırın
- updated tarihini güncelleyin

## PR Akışı

1. Repoyu fork'layın
2. Yeni bir branch açın
3. blocklist.json dosyasını güncelleyin
4. PR açın ve kontrol listesini doldurun
5. CODEOWNERS onayı zorunludur
6. Otomatik kontroller (blocklist doğrulama) geçmelidir

## Notlar

- Büyük toplu değişikliklerde, gerekçeyi PR açıklamasına ekleyin
- Şüpheli hesapları kaynaklayın (örnek: link veya açıklama)
